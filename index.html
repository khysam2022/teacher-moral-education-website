遊戲核心特色

融合教育心理學理論與現代AI技術，提供最真實的道德抉擇體驗

多分支劇情抉擇

在「不處罰任何人」、「只罰受害者」、「處罰所有人」或「輕罰霸凌者」之間抉擇。你的每個決定都會改變故事結局與學生的命運。

寫實3D課室場景

使用 Unity 精心打造的校園環境，包括午休時的課室、課後走廊與操場，搭配角色表情與動作，還原最真實的霸凌現場。

AI 即時評估分析

首創開放式自訂回答！透過 AI (OpenRouter API) 即時評估你輸入的處理方案，深入剖析此舉對受害者、霸凌者及班級氛圍的心理影響。

MiniMax 語音合成

內置 MiniMax Text-to-Audio (T2A) 語音合成引擎，將劇本對話轉為情緒飽滿的廣東話或普通話配音，極大提升劇情的表現力。

學術心理學反饋

遊戲中的反饋由專業文獻支持，包含旁觀者效應、Bowlby 依附理論以及同儕關係研究，讓遊玩過程兼具教育性與學術價值。

雲端數據回報

整合 Google Sheets API 自動上傳玩家的抉擇與反思內容，方便教師培訓機構或學術研究進行群體數據分析與教學討論。

常見問題解答 (FAQ)

為您解答遊玩及技術使用上的常見疑難

本遊戲旨在為在職教師、師範生以及社工提供一個安全、無風險的虛擬環境，去面對和反思校園言語霸凌的處理手法。透過故事分支，讓玩家親身體會不同管理策略對學生產生的深遠心理影響。

當玩家在遊戲中選擇「自行描述處理方式」時，可以輸入高達 50 字的處理對策。遊戲會將文字傳送至 OpenRouter API，由後台 AI 根據教育心理學框架，即時反饋該做法的優缺點與潛在風險。

我們接入了 MiniMax 文本轉語音 (T2A) 語音合成引擎。遊戲會實時將劇本對白轉換成語氣與語調匹配的真人語音，使得霸凌現場的嘲笑聲與受害者的憋屈心聲更加具有臨場感。

遊戲在獲得玩家授權後，會將抉擇數據與反思回答上傳至指定的 Google Sheets。如果您是學校管理員或研究人員，並需要單獨的數據表單，請通過客戶支援與我們聯絡，我們將為您配置專屬的 API 連接。

《老師道德教育》隱私權政策

最近更新日期：2026年6月12日

1. 資訊收集與使用

《老師道德教育》（下稱「本應用程式」）是一個用於教育培訓與模擬的遊戲。我們致力於保護您的隱私。本應用程式僅在以下情況下收集及處理資訊：

  • AI 抉擇文本：當您使用「自行描述處理方式」功能並輸入文字時，該文字會通過安全通道（加密的 HTTPS 連接）發送至 OpenRouter API 以進行語意分析並提供評估。我們不會在 AI 伺服器上永久儲存或關聯您的個人識別資訊。
  • 雲端數據回報：若本應用程式配置了 Google Sheets 數據回報功能，您的抉擇路徑和反思文本會被寫入指定的試算表中。此項功能主要用於學校或培訓機構的教學成效評估，不會主動收集您的設備資訊或個人隱私檔案。

2. 第三方服務提供商

為了實現遊戲的核心功能，我們使用了以下值得信賴的第三方服務：

  • OpenRouter (OpenAI): 用於分析玩家的教育處置回答，並給予基於教育心理學的專業反饋。
  • MiniMax: 用於提供文字轉語音的實時音訊合成服務，為遊戲人物生成語音。

這些第三方服務商僅在提供上述功能所需的範圍內處理傳輸的文字，並遵循其各自的隱私權條款。

3. 數據安全

我們採取了符合業界標準的安全防護措施（如 HTTPS 加密傳輸）來保護您在傳輸過程中的數據安全。然而，請注意，網際網路上的任何傳輸或儲存方法都不是 100% 安全的，我們無法保證其絕對的安全。

4. 兒童隱私

本應用程式主要面向教師、教育研究人員及成人學習者。我們不會故意收集 13 歲以下兒童的個人身份資訊。若您發現有兒童向我們提供了個人資訊，請立即與我們聯絡，我們將採取必要步驟刪除相關數據。

5. 聯絡我們

如果您對本隱私政策有任何疑問，或需要查詢/刪除您上傳的抉擇反思數據，請通過聯絡信箱與我們聯繫：support@teachermoralgame.example.com。
